## Overview

### O Desafio

Os usuários devem ser capazes de:

Ver e explorar um Cartão de Conselhos e ao clicar no botão é gerado um novo número vinculado ao seu Conselho. 
Neste desafio foi utilizado HTML, CSS, Javascript e a API pública que gera conselhos;
Toda Documentação da Api de conselhos: Advice Slip JSON API, pode ser encontrada no https://api.adviceslip.com/
